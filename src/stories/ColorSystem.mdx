A robust, token-based color system for B2B/enterprise UI. Supports light/dark themes, semantic colors, and WCAG-compliant contrast.

## Token Naming Conventions
- `--color-primary`, `--color-secondary`, `--color-tertiary`
- `--color-neutral-100` ... `--color-neutral-900`
- `--color-success`, `--color-info`, `--color-warning`, `--color-error`
- `--color-bg-default`, `--color-bg-muted`, `--color-bg-elevated`, `--color-border`

## Accessibility Considerations
- All color pairs meet or exceed [WCAG AA](https://www.w3.org/WAI/standards-guidelines/wcag/) contrast ratios for text and UI elements.
- Use semantic tokens for status and feedback.
- Test with both light and dark themes.
